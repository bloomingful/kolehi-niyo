## Getting Started

**Note**: Make sure your Flutter environment is setup.
#### Installation

In the command terminal, run the following commands:

    $ git clone https://github.com/emrade/flutter-social.git flutter_social
    $ cd flutter_social/
    $ flutter packages get
    $ flutter run
